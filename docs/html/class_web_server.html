Detailed Description

Represents a single web server that processes one request at a time.

The WebServer tracks whether it's currently busy, how much time is left on its current request, and logs processing events to an external file.

bool WebServer::processRequest ( )

Processes one clock cycle of the current request.

If the server is busy, this function decrements the remaining time. If the request is completed, the server is marked as free and logged in the log file stream.

Returns
True if the server is now free after processing, false otherwise.

If the server is processing a request, this function decrements the remaining time. When the request completes, the server becomes available again and logs the completion event.

Returns
true if the server has completed a request and is now free, false otherwise.
